# 座席表ジェネレーター (Classroom Seat Chart Generator)

このツールは、中学校や高校の教員向けに作成された、A5サイズ縦向きの座席表を簡単に作成・印刷できるウェブアプリケーションです。

## 特徴
- **縦書き対応**: 日本の伝統的な座席表スタイル（氏名・ふりがなを縦書き）。
- **7×7 グリッド**: 最大49席の教室レイアウトに対応。
- **自由な空席設定**: クリッド上で机をクリックするだけで、そこを「空席」として設定。生徒は自動的に詰めて配置されます。
- **縦方向の配置**: 生徒名簿を縦1列目、縦2列目……という順番で流し込み。
- **性別色分け**: 性別データ(F)に基づき、女子生徒の番号を赤色で強調。
- **A5縦印刷最適化**: 1ボタンで最適な印刷用レイアウト（教卓ラベル付き）を表示。

## 使い方
1. **名簿入力**: 画面上部のエリアに名簿を貼り付けます。
   - 形式: `出席番号,姓,名,読み,性別(M/F)`
   - 例: `1,田中,太郎,タナカ,M`
2. **座席の調整**: 下のグリッドで、生徒を座らせたくない席（空席）をクリックして選択します。
3. **印刷**: 「A5サイズで印刷」ボタンを押し、ブラウザの印刷機能を使用して出力します。
   - ※ 印刷設定で「A5」を選択してください。

## 開発・導入方法
このプロジェクトは Vite + React で構築されています。

1. 依存関係のインストール: `npm install`
2. 開発サーバーの起動: `npm run dev`
3. 本番用ビルドの作成: `npm run build`
   - `dist` フォルダに生成された `index.html` と `assets` をサーバーに配置、またはローカルのブラウザで開いて使用できます。

## ディレクトリ構成
- `src/App.jsx`: メインロジック、名簿パース、全体のレイアウト。
- `src/components/SeatGrid.jsx`: 7×7グリッドの構築と生徒の割り当てロジック（Column-Major）。
- `src/components/StudentCard.jsx`: 各座席のカード表示（縦書きスタイル）。
- `dist/`: 配布用のビルド済みファイル。
